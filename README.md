# 1号店移动端前端项目自动化工具
-----

请大家做项目之前，先git下本项目

    git clone https://github.com/TVVT/mobile-start.git

重命名后使用

    mv mobile-start myProject
    
就可以开始你的项目之旅啦！

## 功能：
-----

* 自动编译less为css
* 自动开启source map
* 项目做完，`grunt publish`自动把html、css、js、images复制到`build`文件夹，提供给开发。（css自动压缩）
* chrome安装livereload插件后，启动插件，自动刷新浏览器

## 使用方法
-----

终端里，进入到本文件夹，`grunt`即可
